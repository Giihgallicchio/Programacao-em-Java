# | Java - Estudos e Projetos

Repositório com exercícios e projetos desenvolvidos durante meus estudos de Java, com foco em lógica de programação e Programação Orientada a Objetos (POO).



## | Sobre

Este repositório reúne atividades práticas realizadas durante o aprendizado de Java, abordando desde conceitos básicos até aplicações utilizando Programação Orientada a Objetos.

Os exercícios foram desenvolvidos com o objetivo de fortalecer a lógica de programação e entender a estrutura da linguagem Java.



## | Objetivo

- Aprender a sintaxe do Java  
- Praticar lógica de programação  
- Entender conceitos de POO  
- Desenvolver pequenos sistemas e aplicações  



## | Tecnologias utilizadas

- Java  
- Apache NetBeans 23


## | Conteúdos estudados

Durante os exercícios, foram abordados:

- Estruturas condicionais  
- Estruturas de repetição  
- Vetores (arrays)  
- Métodos  
- Classes e objetos  
- Programação Orientada a Objetos (POO)  



## | Estrutura do Projeto

- `Aula02 - Java POO` → Introdução à POO  
- `Aula03 - Java POO` → Continuação dos conceitos  
- `Aula04 - Java POO` → Exercícios práticos  
- `Aula05 - Java POO` → Aplicação de conceitos  
- `Aula06 - Java POO` → Consolidação do aprendizado  

- `Calculando a Média` → Lógica com cálculos  
- `EquacaoSegundoGrau` → Resolução de equações  
- `Exercicio13` → Exercício prático  
- `ExercicioVetor` → Manipulação de arrays  
- `ProjetoLivro - Java POO` → Projeto aplicando POO  
- `TestaArranjoDeArranjo` → Estruturas mais complexas  
- `UltraEmojiCombat` → Projeto utilizando classes e objetos  
- `ValorPiscina` → Cálculo aplicado  
- `areaCasa` → Cálculo de área  
- `areaCasa2` → Variação do exercício  



## | Destaque

O projeto **UltraEmojiCombat** e o **ProjetoLivro** demonstram a aplicação prática de conceitos de Programação Orientada a Objetos.



## | Evolução

Este repositório representa minha evolução no aprendizado de Java, desde exercícios básicos até projetos mais estruturados utilizando POO.
